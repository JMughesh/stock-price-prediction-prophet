# stock-price-prediction-prophet
This project demonstrates how to use Facebook Prophet, a powerful open-source forecasting tool developed by Meta, to predict stock prices for ITC Limited (ITC.NS). Prophet is particularly effective for time series data with strong trends and seasonality.
